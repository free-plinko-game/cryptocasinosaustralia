### Introduction to Ripple

Ripple, commonly referred to by its cryptocurrency token XRP, is a real-time gross settlement system, currency exchange, and remittance network created by Ripple Labs Inc. Founded in 2012, Ripple aims to enable instant, secure, and nearly free global financial transactions of any size with no chargebacks. The Ripple protocol is built upon a distributed open-source Internet protocol, consensus ledger, and native cryptocurrency identified as XRP.
